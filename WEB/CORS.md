# CORS란?

- Cross-Origin Resource Sharing의 줄임말입니다.
- 해석해보면 `교차 출처 자원 공유`가 됩니다.

<br>
<br>

# CORS ERROR란?

- 브라우저 간에 데이터를 주고 받는 과정에서, 도메인 이름이 서로 다른 사이트 사이에서 API 요청을 할 경우, 사이트 간에 공유를 설정하지 않았다면 CORS 에러가 발생합니다.
